# 🛒 Zepto SQL Project

## 📌 Overview
This project focuses on analyzing a dataset using SQL to extract meaningful business insights. It demonstrates strong SQL skills including data cleaning, transformation, and analytical querying.

The goal of this project is to simulate real-world data analysis scenarios and answer key business questions using structured query language (SQL).

---

## 🎯 Objectives
- Perform data cleaning and preprocessing
- Analyze business metrics using SQL queries
- Generate actionable insights from raw data
- Practice advanced SQL concepts in a real dataset

---

## 🧰 Tools & Technologies
- SQL
- PostgreSQL
- Database concepts (Joins, Aggregations, Subqueries)

---

## 📊 Key Features
- Data filtering and cleaning
- Use of JOINs to combine multiple tables
- Aggregations (SUM, AVG, COUNT, etc.)
- Subqueries and nested queries
- Business insight generation
